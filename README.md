## What this is

Pocket Money Hero is an HTML5 and JavaScript educational game made to run in a web browser. Its core focus is practical financial awareness, helping players learn budgeting, money handling, and setting goals by working through everyday spending decisions.

The experience mixes interactive rounds with random life events and a progress dashboard so players can see how choices affect outcomes over time. It is aimed at student learning and hackathon-style demos, and the layout is responsive so it can adapt to different screen sizes and web devices.
